# LampStore — Интернет-магазин ламп

React-приложение интернет-магазина с каталогом товаров, корзиной и оформлением заказа.

## Возможности

- Каталог товаров с поиском и фильтрацией по категориям
- Страница детального просмотра товара с характеристиками
- Корзина с управлением количеством и удалением товаров
- Оформление заказа с формой данных покупателя
- Адаптивный дизайн от 320px

## Требования

- [Node.js](https://nodejs.org/) версии 18 или выше
- npm (входит в состав Node.js)

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone <url-репозитория>
cd <папка-проекта>
```

### 2. Установить зависимости

```bash
npm install
```

### 3. Запустить в режиме разработки

```bash
npm run dev
```

Приложение откроется по адресу: [http://localhost:5000](http://localhost:5000)

### 4. Сборка для продакшена

```bash
npm run build
```

Собранные файлы появятся в папке `dist/`.

### 5. Предпросмотр собранной версии

```bash
npm run preview
```

## Структура проекта

```
├── index.html              # Точка входа HTML
├── vite.config.js          # Конфигурация Vite
├── package.json            # Зависимости и скрипты
└── src/
    ├── main.jsx            # Точка входа React
    ├── App.jsx             # Корневой компонент и маршруты
    ├── App.css             # Глобальные стили
    ├── components/
    │   ├── Layout.jsx      # Шапка и обёртка страниц
    │   └── ProductCard.jsx # Карточка товара
    ├── context/
    │   └── CartContext.jsx # Состояние корзины (Context API)
    ├── data/
    │   └── products.js     # Данные о товарах
    └── pages/
        ├── Catalog.jsx         # Каталог с поиском и фильтрами
        ├── ProductDetail.jsx   # Страница товара
        ├── Cart.jsx            # Корзина
        ├── Checkout.jsx        # Оформление заказа
        └── OrderConfirmation.jsx # Подтверждение заказа
```

## Технологии

| Технология | Версия | Назначение |
|---|---|---|
| React | 18 | UI-фреймворк |
| React Router DOM | 6 | Клиентская маршрутизация |
| Vite | 5 | Сборщик и dev-сервер |

## Маршруты

| Путь | Страница |
|---|---|
| `/` | Каталог товаров |
| `/product/:id` | Страница товара |
| `/cart` | Корзина |
| `/checkout` | Оформление заказа |
| `/order-confirmation` | Подтверждение заказа |
